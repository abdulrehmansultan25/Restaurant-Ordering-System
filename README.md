# 🍔 Food Hub - OOP Restaurant Ordering System

## 📌 Project Overview

Food Hub is a Python-based Restaurant Ordering System developed using **Object-Oriented Programming (OOP)** concepts and **Streamlit**. The application allows users to browse a restaurant menu, customize food items, place orders, choose different payment methods, and track order status through an interactive interface.

This project was developed as an academic OOP Lab Project to demonstrate the practical implementation of major Object-Oriented Programming principles and design patterns.

## 🚀 Features

- Interactive Restaurant Menu
- Shopping Cart Management
- Quantity Update & Item Removal
- Food Customization (Extra Cheese)
- Multiple Payment Methods (Cash, Card, EasyPaisa)
- Order Tracking
- OOP Assessment Dashboard
- Live System Logs
- Modern Streamlit UI

## 🛠 Technologies Used

- Python 3.x
- Streamlit
- Object-Oriented Programming (OOP)

## 📚 OOP Concepts

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Singleton Pattern
- Strategy Pattern
